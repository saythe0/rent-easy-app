# RentEase

Проект RentEase создан в рамках учебной практики 2025 года (4 курс).

## Описание проекта

RentEase — это веб-платформа для онлайн-аренды вещей между пользователями. Сайт предназначен для удобного поиска, публикации и бронирования товаров для временного использования: инструментов, бытовой техники, электроники и другого оборудования. Проект решает задачу упрощения процесса аренды без необходимости покупки вещей.

## О коде

Проект построен на Laravel v12 с REST API и админ-панелью на Filament v4. Основная логика сосредоточена в моделях, контроллерах и ресурсах API. Для хранения данных используется MySQL. Локальная разработка и запуск выполняются через Laravel Sail (Docker).

## Установка

1. Склонировать репозиторий.
2. Установить зависимости:
   - `composer install`
3. Поднять окружение через Sail:
   - `./vendor/bin/sail up -d`
4. Сгенерировать ключ приложения:
   - `./vendor/bin/sail artisan key:generate`
5. Выполнить миграции:
   - `./vendor/bin/sail artisan migrate`

## Запуск

Для запуска локально используйте:

```
./vendor/bin/sail up -d
```

Остановка:

```
./vendor/bin/sail down
```

## .env

Скопируйте файл окружения и заполните переменные:

```
cp .env.example .env
```

Основные параметры:

- `APP_NAME=RentEase`
- `APP_URL=http://localhost`
- `DB_CONNECTION=mysql`
- `DB_HOST=mysql`
- `DB_PORT=3306`
- `DB_DATABASE=rent_ease`
- `DB_USERNAME=sail`
- `DB_PASSWORD=password`
